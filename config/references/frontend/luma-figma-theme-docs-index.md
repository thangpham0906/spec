# Luma Figma Theme Docs Index
> Bộ tài liệu chuẩn để team triển khai theme Magento Luma từ Figma (desktop + mobile) theo XML + BEM + LESS.

## Dùng khi nào

- Bắt đầu page mới (homepage, PLP, PDP, cart, checkout).
- Onboard dev mới vào frontend theme.
- Review chất lượng trước khi merge.

## Cách dùng nhanh

1. Đọc architecture để khóa nguyên tắc.
2. Map token + breakpoint trước khi code.
3. Chốt component contract theo template.
4. Đi theo workflow từng bước.
5. Chỉ merge khi pass QA gate.
6. Với section có CMS content seed, áp dụng section-module pattern.
